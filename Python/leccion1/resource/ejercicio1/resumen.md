## Tipo de variable
#

        - int = número entero
        - float = número decimal
        - String = Cadena de caracteres
        - Bool = variable Booleana
    
    * Cuando utilizamos la función "type()" obtenemos 
    - la información de que tipo de variable ingresamos
dentro de los paréntesis de la función.
